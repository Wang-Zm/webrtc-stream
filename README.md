## 目录
* H5 语音识别
* Vue 组件间通信
* video 标签之间的流传输
* WebRTC 本地对等连接
* WebRTC 网络对等连接
* WebRTC + Canvas 实现一个双人协作的共享画板
## 计划

* WebRTC 实战（三）：实现一个一对一的视频聊天项目，包括但不限于截图、录制等。
